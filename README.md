# Ionic Angular Jest Seed
An Ionic Angular project with Jest as testing framework

## Env Engine

Node 10.20.1
```bash
nvm use
```

## Installation

```bash
npm install
```

## Format code

```bash
npm run format
```

## Testing

Test watcher
```bash
npm test
```

Code coverage report
```bash
npm test:coverage
```
